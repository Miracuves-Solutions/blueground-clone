## How It Works

1. Guest browses property listings with photos, amenities, and availability
2. Guest selects dates and books with flexible lease terms
3. Host receives booking confirmation and manages check-in details
4. Tenant portal provides self-service for rent payments and maintenance
5. Platform processes payments and escrows security deposits
6. Property management dashboard tracks occupancy, revenue, and maintenance

---

## Core Features

### Guest App
- Browse property listings with photos, amenities, and virtual tours
- Search and filter by city, dates, price, property type
- Flexible lease booking (30+ days) with instant confirmation

- Booking history and receipt downloads

- Property listing management with availability calendar

### Admin Panel
- Property network and portfolio management

    A[Flutter Apps<br/>Android · iOS] --> G[REST API<br/>Laravel Backend]
    W[Web Platform<br/>Responsive] --> G
    P[Partner Panel] --> G
    AD[Admin Dashboard] --> G
    G --> DB[(MySQL)]

**Stack:**

| Layer | Technology |
|---|---|
| Mobile Apps | Flutter (iOS + Android, single codebase) |
| Backend API | Node.js + Express |
| Database | MongoDB |
| Payments | Stripe, Razorpay, PayPal |
| Maps | Google Maps API |
